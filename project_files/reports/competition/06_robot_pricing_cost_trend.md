# 机器人整机价格、成本与未来三年走势分析

生成日期：2026-05-26

## 1. 管理层结论

本专题只覆盖具身智能整机：人形、四足、轮式双臂、移动操作和服务平台，不纳入零部件、纯软件平台、传感器和单独配件。完整结构化清单见 [机器人整机价格清单](../../data/robot_pricing.csv)。

1. 公开标价主要出现在开发者、科研教育和早期家庭机器人产品上，典型价格从数千美元到十万美元不等。工业级项目仍以询价、试点、RaaS 或合同制为主。
2. 低价不等于可落地。Unitree R1/G1、EngineAI SA01/PM01、LimX TRON 1 降低了硬件门槛，但主要价值仍是研发、教学、训练和场景验证。
3. 真正影响企业采购的不是裸机价，而是总拥有成本：部署、软件、远程支持、备件、安全责任、停机和运维组织会决定 ROI。
4. 未来三年价格会分层下降：开发者平台降得最快，工业整机降得较慢，成熟场景会更多转向订阅、RaaS 或按任务结果收费。
5. 管理层不能用“官网有价格”直接推导“可规模部署”。官网价格适合作为预算锚点，正式试点仍必须要求供应商给出任务、场地、运维和责任边界报价。

## 4. 成本结构分析

机器人整机成本不应只按 BOM 看。管理层做预算时至少拆成八个成本池：

| 成本池 | 说明 | 管理层要问的问题 |
| --- | --- | --- |
| 本体硬件 | 机身结构、关节模组、执行器、线束、外壳 | 关键部件是否自研，寿命和备件周期是多少 |
| 执行器/关节 | 人形和四足成本核心，影响负载、速度和耐久 | 关节失效后更换成本和交付周期是多少 |
| 传感器与算力 | 相机、激光雷达、深度传感、边缘计算模块 | 是否满足本场景光照、遮挡、网络和安全要求 |
| 电池与结构件 | 电池、快换、充电、底盘、防护结构 | 连续工作时长、充电方式和电池更换成本是多少 |
| 软件/数据/遥操作 | VLA、调度系统、远程接管、数据闭环 | 是否额外收费，数据归属和模型改进权归谁 |
| 部署集成 | 场地改造、网络、接口、SOP、员工培训 | 首点部署和第二点复制成本分别是多少 |
| 运维备件 | 巡检、保养、维修、远程支持、驻场服务 | SLA、停机补偿、备件库存和响应时间怎么约定 |
| 风险责任 | 安全、保险、隐私、碰撞、劳动和合规责任 | 发生事故或数据问题时责任如何划分 |

结论是：官网裸机价只能回答“买一台大约多少钱”，不能回答“上线一个业务任务一年多少钱”。企业做试点时，应要求供应商同时给出买断价、租赁价、RaaS 价、运维价和扩点价。

## 5. 未来三年价格走势

### 5.1 2026-2027：低价平台继续下探

开发者和科研教育平台会继续降价，原因是中国供应链、人形轻量化方案、标准化关节和线上直销会压低入门门槛。5-15 万元人民币的人形或双足平台会变多，但多数仍服务于教学、研发、演示和早期数据采集。

### 5.2 2027-2028：工业试点报价分化

工业人形、工业四足和轮式双臂平台不会简单按消费电子曲线降价。价格会按场景分化：标准硬件可能下降，但安全认证、末端工具、客户流程接口、驻场运维和任务软件会变成主要报价项。管理层看到的合同价格可能不降反升，因为报价从“机器”转向“可用系统”。

### 5.3 2028-2029：RaaS 与按结果付费增强

如果仓储、巡检、零售盘点、制造搬运等任务验证成功，供应商会更倾向 RaaS、订阅、按小时或按任务计费。买断价格仍存在，但大客户会更关注单位任务成本、可用率和复制成本。到 2029 年，整机价格的战略意义会下降，运营成本和任务成功率的战略意义会上升。

## 6. 管理层采购和试点建议

1. 用公开价做预算锚点，不用公开价做采购结论。公开价适合判断行业门槛，不能替代真实项目报价。
2. 对低价人形平台，优先用于内部学习、研发验证和数据采集，不要包装成立刻降本。
3. 对工业人形和四足巡检，要求供应商拆分设备、软件、部署、运维和备件报价，并给出 90 天试点 SLA。
4. 对 RaaS 和订阅模式，重点比较单位任务成本、可用率、远程接管比例、最小合同期和退出成本。
5. 对官网未披露价格的主流公司，不应视为“不成熟”，更可能是产品处于 B2B 试点、定制化或大客户合同阶段。
6. 每季度更新价格清单，把“官网标价变化、是否开始公开售价、是否从买断转向订阅/RaaS”作为行业成熟度信号。
