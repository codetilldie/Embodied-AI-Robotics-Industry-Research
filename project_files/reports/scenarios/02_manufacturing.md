# 场景报告：制造业与汽车产线

生成日期：2026-05-26

## 1. 成熟度判断

制造业是人形机器人最重要的真实验证场，但当前总体处于 L1-L2：已有客户试点和小批量合同，但距离规模化替代仍有距离。

## 2. 适合任务

- 料箱搬运。
- SPS 分拣。
- 产线边供料。
- 低节拍装配辅助。
- 质量检查。
- 夜间巡检。
- 危险或重复岗位辅助。

## 3. 为什么适合试点

制造业有明确 SOP、成本核算和安全管理体系。相比家庭和开放服务场景，工厂任务更容易被限定，客户也更愿意为效率和安全付费。

## 4. 主要瓶颈

- 可靠性和连续工作时间。
- 工业安全认证。
- 与产线节拍匹配。
- 夹具、料箱和工位标准化。
- 与 MES/WMS/ERP 系统连接。

## 5. 供应商类型

| 类型 | 代表 | 适合切入 |
| --- | --- | --- |
| 人形机器人 | 优必选、智元、Figure、Apptronik | 工位辅助、搬运、分拣 |
| AMR | 斯坦德、优艾智合、Amazon Robotics | 厂内物流 |
| 工业视觉 | 梅卡曼德、Covariant | 抓取、检测、上下料 |
| 协作机械臂 | 节卡、遨博、越疆、非夕 | 固定工位操作 |

## 6. 管理层试点建议

第一阶段不要让机器人进入核心生产节拍。建议选择边界清晰、低安全风险、低节拍的辅助任务，例如空箱搬运、来料分拣、产线边补料和夜间巡检。

## 7. 成功指标

- 单班次有效工作小时。
- 单任务成功率。
- 人工接管次数。
- 停机时间。
- 单次任务成本。
- 员工安全事件。
- 复制到其他工位的改造成本。
