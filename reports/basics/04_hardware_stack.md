# 行业基础课：机器人硬件栈

生成日期：2026-05-26

## 1. 硬件决定成本下限和可靠性上限

具身智能机器人不是纯软件产品。即使模型进步很快，机器人仍受限于电机、减速器、关节、传感器、电池、散热、结构件、制造工艺和维护体系。

## 2. 核心硬件模块

| 模块 | 作用 | 管理层关注点 |
| --- | --- | --- |
| 关节模组 | 决定运动能力和负载 | 寿命、成本、可维护性 |
| 电机/驱动 | 提供动力 | 能耗、响应速度、稳定性 |
| 减速器 | 提升扭矩和精度 | 国产替代、成本、寿命 |
| 传感器 | 感知环境和力反馈 | 可靠性、隐私、误报 |
| 灵巧手 | 完成抓取和操作 | 成本、耐久、触觉 |
| 电池 | 决定续航 | 安全、换电、充电效率 |
| 计算平台 | 运行感知和模型 | 算力、功耗、成本 |
| 结构件 | 承担外形和安全 | 重量、强度、维护 |

## 3. 中国供应链机会

中国在电机、结构件、制造、传感器、服务机器人和整机组装方面具备优势。随着人形机器人公司放量，关节模组、灵巧手、力传感器、减速器、线束、热管理和电池系统都有机会形成新的供应链需求。

## 4. 管理层选型影响

企业采购机器人时，不应只看演示效果，还要问：

- 核心部件是否可替换。
- 备件周期多长。
- 关节寿命和维护成本是多少。
- 电池是否支持快换或自动回充。
- 是否有本地服务团队。
- 关键零部件是否依赖单一供应商。

## 5. 成本下降路径

成本下降通常来自：

1. 核心零部件国产化。
2. 设计平台化。
3. 规模制造。
4. 软件复用。
5. 维护流程标准化。
6. 任务边界收敛，避免为“全能”付出过高成本。
