# 战略决策工具：ROI 测算模型

生成日期：2026-05-26

## 1. ROI 不只看设备价格

机器人项目总成本包括设备、软件、部署、场地改造、运维、培训、备件、停机和管理成本。

## 2. 成本项

| 成本 | 说明 |
| --- | --- |
| 设备成本 | 买断价或租赁费 |
| 软件成本 | 平台、算法、调度、升级 |
| 部署成本 | 场地、网络、充电、接口 |
| 运维成本 | 维护、备件、驻场、远程支持 |
| 培训成本 | 员工培训和 SOP 调整 |
| 风险成本 | 停机、安全、保险、责任 |

## 3. 收益项

| 收益 | 说明 |
| --- | --- |
| 节省工时 | 替代或减少重复劳动 |
| 提升效率 | 更高处理量和更少等待 |
| 降低风险 | 减少危险区域人工进入 |
| 提升服务 | 更稳定的配送、清洁、巡检 |
| 数据价值 | 形成可追溯运营数据 |

## 4. 简化公式

年度净收益 = 年度可量化收益 - 年度总成本。

回本周期 = 一次性投入 / 月度净收益。

如果收益无法量化，必须把项目归类为“品牌体验”或“战略探索”，不能包装成降本项目。

## 5. 试点期 ROI 记录

试点期间至少记录：

- 每天任务数。
- 单任务人工用时。
- 机器人任务成功率。
- 人工接管次数。
- 故障和停机时长。
- 运维响应时间。
- 员工实际使用率。

## 6. 管理层判断

如果一个项目无法在 90 天内记录基本任务数据，就不应进入扩大部署。
