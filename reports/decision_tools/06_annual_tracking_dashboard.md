# 战略决策工具：年度跟踪看板

生成日期：2026-05-26

## 1. 看板目的

年度跟踪看板用于把行业热度转化为可复盘的管理动作。建议每月更新一次，每季度做一次管理层复盘。

## 2. 跟踪维度

| 维度 | 指标 |
| --- | --- |
| 公司 | 新客户、新订单、新产品、新融资 |
| 技术 | VLA、灵巧手、触觉、仿真、数据平台 |
| 场景 | 制造、物流、巡检、零售、服务、康养 |
| 价格 | 单机价格、租赁费、运维费 |
| 政策 | 国家、地方、标准、补贴 |
| 风险 | 安全事故、夸大宣传、客户流失 |

## 3. 月度问题

1. 本月是否出现具名客户案例。
2. 本月是否出现付费订单或采购合同。
3. 本月是否有可复用任务，而不是单次 Demo。
4. 本月是否有价格下降信号。
5. 本月是否有监管或安全事件。

## 4. 季度复盘

每季度重新回答：

- 哪些场景成熟度上升。
- 哪些供应商值得进入访谈名单。
- 是否启动或调整试点。
- 是否需要预算或组织准备。
- 是否有风险需要暂停关注。

## 5. 输出形式

建议每季度形成一页 Markdown 复盘：

- 本季度最重要 5 条事实。
- 对公司战略的 3 条影响。
- 下季度要跟踪的 5 个问题。
- 是否建议进入试点。
