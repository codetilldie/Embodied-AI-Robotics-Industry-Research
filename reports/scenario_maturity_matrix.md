# 场景成熟度矩阵

生成日期：2026-05-26

## 1. 评估口径

本矩阵用于判断具身智能机器人在不同场景的落地优先级。评分不是市场规模排名，而是综合技术可行性、商业闭环、客户付费意愿、部署复杂度和证据强度后的初版判断。

成熟度等级沿用项目口径：

- L0：概念演示。
- L1：真实场景试点。
- L2：小批量交付。
- L3：规模化商用。

## 3. 场景分析

### 3.1 仓储物流/电商履约

**成熟度判断：L3。**

仓储物流是当前机器人商业化最确定的场景之一。核心原因是环境相对可控、任务高频、客户具备自动化预算，并且效率提升可以直接映射到履约成本。Amazon Robotics 的履约中心机器人体系已经是百万级内部部署，Simbe、Covariant、Agility 等公司也分别从库存识别、机器人抓取和人形搬运切入。

人形机器人在仓储中的机会主要不是替代所有专用设备，而是补足传统自动化难以覆盖的“人类空间任务”，例如在既有仓库中搬箱、补位、处理异常和衔接不同自动化岛。

**关键跟踪指标：**

- 是否有明确付费合同或 RaaS 收入。
- 单台机器人日均工作小时数。
- 与人工相比的单位任务成本。
- 故障率、维护成本和安全隔离要求。
- 是否能从单客户复制到多客户。

### 3.2 汽车制造/工业产线

**成熟度判断：L1-L2。**

汽车制造是人形机器人最容易被验证的高价值场景之一。BMW 对 Figure 02、Mercedes-Benz 对 Apptronik Apollo、优必选 Walker S 系列进入工业产线任务，说明主机厂愿意拿真实产线任务验证人形机器人的可用性。

但该场景不应被过早判断为规模化。汽车产线对安全、节拍、可靠性、维护和工艺一致性要求极高。当前公开案例多集中在零件搬运、料箱处理、分拣、质检、SPS 供料等有限任务，离全流程替代还有明显距离。

**关键跟踪指标：**

- 机器人是否进入正式生产节拍。
- 是否连续运行多班次。
- 是否跨工位复用。
- 客户是否扩大采购。
- 是否完成安全认证和工厂 IT/OT 系统集成。

### 3.3 零售门店/货架智能

**成熟度判断：L2-L3。**

零售场景需要拆成两层看：货架扫描、库存盘点和促销检查已经较成熟，Simbe Tally 是代表；机器人主动取货、补货、交付和无人零售仍处于 L1-L2 之间，Galbot G1 等公司正在探索。

零售场景的优势是任务高频、空间相对半结构化、店内商品 SKU 有数据基础。难点在于开放环境中的顾客安全、商品遮挡、货架变化、补货动作稳定性和门店运维。

**关键跟踪指标：**

- 是否能每天稳定巡店或取货。
- SKU 识别准确率和缺货识别准确率。
- 门店员工是否愿意使用机器人输出。
- 是否从试点店扩展到连锁门店。
- 是否产生直接收入或减少缺货损失。

### 3.4 能源/化工/电力巡检

**成熟度判断：L2-L3。**

巡检是四足机器人和工业 AMR 的强场景。ANYbotics、Deep Robotics、Youibot 等公司的产品已经能在电力、化工、矿业、半导体等复杂或高风险区域承担巡检任务。相比人形机器人，四足形态对楼梯、栅格、碎石、坡道等地形更合适，任务边界也更清晰。

这一场景的商业价值来自减少危险岗位暴露、提升巡检频率和数据可追溯性。它更像“移动传感平台+AI分析系统”，不一定需要复杂灵巧操作。

**关键跟踪指标：**

- 巡检任务是否从人工替代转为常态化排班。
- 识别准确率、漏检率和误报率。
- 机器人在复杂地形中的可用率。
- 是否支持自动回充和远程调度。
- 是否进入客户安全生产体系。

### 3.5 家庭服务/康养陪护

**成熟度判断：L0-L1。**

家庭服务是长期空间最大的场景，但短期成熟度最低。家庭环境高度开放，任务差异巨大，安全责任复杂，用户对价格、隐私和故障容忍度很低。1X NEO、NEURA 4NE1、Astribot S1 等产品展示了方向，但真实家庭中的长期可用性仍需要验证。

家庭场景更可能先从“远程协助+有限自主任务+高频轻服务”开始，而不是一开始就实现全自主家务机器人。

**关键跟踪指标：**

- 是否进入真实家庭并持续使用超过 3 个月。
- 用户是否愿意为订阅或维护付费。
- 遥操作介入比例。
- 家务任务成功率和安全事故记录。
- 隐私、摄像头、家庭数据的合规处理。

## 4. 当前优先级建议

| 优先级 | 场景 | 理由 |
| --- | --- | --- |
| P0 | 仓储物流/电商履约 | 商业闭环最清晰，已有规模化标杆 |
| P0 | 能源/化工/电力巡检 | 任务边界明确，客户痛点强，四足/AMR已落地 |
| P1 | 汽车制造/工业产线 | 人形机器人最重要的验证场，但仍需观察规模化 |
| P1 | 零售门店/货架智能 | 货架智能成熟，操作型零售机器人值得持续跟踪 |
| P2 | 家庭服务/康养陪护 | 长期价值大，短期技术和商业不确定性高 |
